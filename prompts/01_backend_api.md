# Prompt – Backend API MHEALTH

Prompt utilizado para diseñar la API del proyecto final:

"Necesito una API en FastAPI para reconocimiento de actividad humana usando el dataset MHEALTH.
Debe tener un endpoint /health para verificación y un endpoint /detect que reciba archivos .log,
procese los datos con un pipeline y devuelva la actividad predominante, distribución de actividades
y si el patrón es anómalo."

Uso:
Diseño de la estructura del backend, endpoints y formato de respuesta JSON.
