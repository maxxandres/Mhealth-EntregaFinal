# Prompt – Frontend UI

Prompt utilizado para diseñar la interfaz web:

"Quiero un frontend simple en HTML/CSS/JavaScript (Vue 3 CDN) que permita subir un archivo .log,
consumir una API FastAPI y mostrar resultados de reconocimiento de actividad humana de forma clara
para un usuario no técnico."

Uso:
Diseño de la interfaz, carga de archivos, visualización de resultados y distribución de actividades.
